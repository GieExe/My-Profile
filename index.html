Barnyard Defenders — Full Project Details

Barnyard Defenders is a high-fidelity, animal-themed Tower Defense (TD) web application combining classic strategy gameplay with modern Gacha collection mechanics and a progressive upgrade system.

Core Gameplay Loop

Prevent waves of invasive insects from reaching the homestead by strategically placing animal defenders along a winding path.

Key Systems

  • Gacha Summoning: Spend "Acorns" to pull random units, forcing adaptive strategy.
  • 1x & 5x Summons: Single or bulk pulls for faster army building.
  • Rarity Variants: Normal (89%), Golden (10% — 1.5x stats), Rainbow (1% — 2.5x stats + RGB visuals).
  • Boss Waves: Every 5th wave features a high-HP "Big Bad Wolf" boss.
  • Economy: Balance acorns between new summons, Level 3 upgrades, or selling weak variants.

Technical Specs

  • Single-file vanilla JS + HTML5 Canvas + Tailwind CSS.
  • Particle systems, dynamic shadows, bobbing animations, rotation logic.
  • 2x Fast Forward mode with consistent frame rate.
  • Fully responsive with touch-friendly controls.

Defender Roster

  • Common: Chicken (Rapid Fire), Sheep (Balanced).
  • Rare: Pig (Heavy Hitter), Dog (High Speed).
  • Epic / Legendary: Cow (Sniper), Horse (Extreme Performance).
